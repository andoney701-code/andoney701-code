## Current Work

### Kustika

An event management platform built to support the complete lifecycle of an event.

**Key features**

- Authentication with JWT
- Role-Based Access Control (RBAC)
- Event management
- Ticket sales
- QR ticket validation
- Administrative dashboard

- RESTful APIs
- PostgreSQL database
- Google Cloud deployment
